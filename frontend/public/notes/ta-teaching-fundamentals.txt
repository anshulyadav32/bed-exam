Teaching Aptitude Notes: Teaching Fundamentals
===============================================

TEACHING METHODS
- Lecture method: teacher-centered, good for large groups, limited interaction
- Discussion method: student participation, critical thinking
- Demonstration method: "show and tell", effective for practical skills
- Project method: Kilpatrick's approach, learning by doing
- Problem-solving method: Dewey's approach, develops higher-order thinking
- Cooperative learning: group work, peer teaching

LEARNING PROCESS
- Bloom's Taxonomy (Cognitive domain): Knowledge → Comprehension → Application → Analysis → Synthesis → Evaluation
- Revised Bloom's: Remember → Understand → Apply → Analyze → Evaluate → Create
- Affective domain (Krathwohl): Receiving → Responding → Valuing → Organizing → Characterizing
- Psychomotor domain (Simpson): Perception → Set → Guided Response → Mechanism → Complex Response

LESSON PLANNING
- Herbart's 5 Steps: Preparation, Presentation, Association, Generalization, Application
- Components: Aims, Previous Knowledge, Materials, Introduction, Development, Recapitulation, Assignment
- Types: Unit plan, Daily lesson plan, Annual plan

CLASSROOM COMMUNICATION
- One-way: teacher → student (lecture)
- Two-way: interactive (Q&A, discussion)
- Non-verbal communication: body language, eye contact, gestures
- Feedback: immediate correction improves learning retention

EXAM TIPS
- Bloom's Taxonomy levels are frequently tested
- Know characteristics of each teaching method
- Herbart's steps sequence must be memorised
